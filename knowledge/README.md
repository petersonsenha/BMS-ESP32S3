# Knowledge Vault da BMS

Esta pasta contem a camada de conhecimento do projeto para o Obsidian.

## Como usar

1. abra a raiz do repositorio no Obsidian
2. navegue pela pasta `knowledge/`
3. comece pelos arquivos em `00-MOCs/`
4. abra os mapas em `90-Canvas/`
5. use `Graph view` para enxergar as relacoes entre as notas
